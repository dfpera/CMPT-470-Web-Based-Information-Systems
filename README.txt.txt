Hosted @ http://localhost:3000/


Noted:

Login:

- Sign Up: user will add a Username, Name, and Password in order to create a new account.

- Sign In: user can use a previously created account to gain access to the application.


Notes:

- New Note: user can create a new note using the '+Note' button in the top-left corner.
A title, text, and tag can be entered and stored by the system.

- View Notes: notes are all displayed on the left-hand side of the application.

- Tags: The right-side of the application displays notes categorized by their tag.
A large box containing a specific tag contains all notes which contain that tag.
